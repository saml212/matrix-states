# NeurReps 2026 EA — submission bundle

`neurreps-ea-submission.tex` is the flattened single-file source the
NeurReps CFP asks for (built by `make bundle` from the modular tree one
level up); `neurreps-ea-submission.pdf` is its compiled review build
(anonymized: the jmlr template omits the author block during review).
`jmlr.cls`/`jmlrutils.sty` come verbatim from the workshop's official
style zip.

## Figure provenance

Both figures are generated by `../figures/figure_gen.py`, which loads
ONLY archived raw result files and asserts the md5 of every source
before plotting (build fails loudly if an archive changed). Sources and
their recorded md5s are pinned in that script's `SOURCE_MD5` dict:

- `fig1_razor_step.pdf` — 20 zero-pad cell JSONs from
  `experiment-runs/2026-07-09_m3fix_harvest/` + 12 from
  `experiment-runs/2026-07-09_m3fix_s3ext/` (field
  `crosscheck_recovered_frac_90`).
- `fig2_rank_tracking.pdf` — the 19 unconstrained-cell JSONs from
  `experiment-runs/2026-07-09_capability_sweep_harvest/results/`
  (field `restricted_effective_rank`).
