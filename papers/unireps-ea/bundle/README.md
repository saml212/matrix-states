# UniReps 2026 EA — submission bundle

`unireps-ea-submission.tex` is the flattened single-file source (built
by `make bundle` from the modular tree one level up);
`unireps-ea-submission.pdf` is its compiled anonymized submission build
(NeurIPS-template default mode: anonymous block + line numbers, per the
UniReps CFP). `neurips.sty` is the official NeurIPS kit (2025 stand-in
until the 2026 styles ship — see `../VENUE_REQUIREMENTS.md`).

## Figure provenance

Both figures are generated by `../figures/figure_gen.py`, which loads
ONLY archived raw result files and asserts the md5 of every source
before plotting (build fails loudly if an archive changed). Sources and
their recorded md5s are pinned in that script's `SOURCE_MD5` dict:

- `fig1_convergence.pdf` — the 19 unconstrained-cell JSONs from
  `experiment-runs/2026-07-09_capability_sweep_harvest/results/`
  (field `restricted_effective_rank`); the inset's mean difference and
  TOST margin are recomputed from the same S4/A5 per-seed values.
- `fig2_razor_step.pdf` — 20 zero-pad cell JSONs from
  `experiment-runs/2026-07-09_m3fix_harvest/` + 12 from
  `experiment-runs/2026-07-09_m3fix_s3ext/` (field
  `crosscheck_recovered_frac_90`).

Non-figure numerical claims trace to evidence rows U1–U7 in
`../brief.md`, each carrying its verdict record, raw artifact path, and
md5. Additional single-file artifacts quoted in the text:

- `harvest_summary.json` md5 `7dce77dcba724cd1004419ac71fe5f2f` (U1/U2/U6)
- sweep `harvest_analysis_output.txt` md5
  `854a4bd7c46e626badcc0fbf05d0e07a` (U5/U6)
- `gate1b_recheck.txt` md5 `2d170cc03011cc56105adeae9929e481` (U7)
