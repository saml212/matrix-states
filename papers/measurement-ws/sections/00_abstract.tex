\begin{abstract}
Empirical claims about learned systems inherit the health and
calibration of the instruments that measure them. We report six pre-publication incidents from one
measurement-heavy research program
on fast-weight sequence models in which a trained model appeared to
fail and the failure traced instead to the instrument: an admission
tolerance calibrated on the wrong key population; a probe reading a
layer causal state-zeroing later proved inert; an identity-gauge
assumption measured on one architecture and silently carried to
another; an uncentered
covariance estimator degenerate on the task's own near-orthogonal
targets; a target whose ambient identity block taxed every rank
budget; and a cross-check reference invoked on the transposed side of
a state convention. Each incident was caught by one and the same
discipline:
instrument-health adjudication before any model verdict,
pre-registered crosschecks graded where the primary lens's assumptions
fail, falsifier controls run to completion, and raw-artifact
tiebreaks. The discipline is itself falsifiable; its decisive
crosscheck survived a pre-registered shuffled-target control that
would have voided the adjudication. It does not launder results: the
corrected lens left the pre-registered endpoint verdict standing and
two genuine model failures intact. We argue the discipline transfers
to any empirical evaluation gated by numerical checks, learned probes,
or basis-dependent readouts.
\end{abstract}
