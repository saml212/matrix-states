# NeurReps 2026 EA ŌĆö submission bundle ("When the Gradient Sees Rank")

`rank-recruitment-ws-submission.tex` is the flattened single-file source
the NeurReps CFP asks for (built by `make bundle` from the modular tree
one level up); `rank-recruitment-ws-submission.pdf` is its compiled

during review; verified byte-identical to `../main.pdf` and confirmed to
compile standalone from this directory). `jmlr.cls`/`jmlrutils.sty`
come verbatim from the workshop's official style zip.
