## Before hitting submit

1. **Compile locally** with `make` in this directory. Check:
   - PDF is 8 pages or fewer excluding references
   - All citations resolve (no `??` markers)
   - All figure refs resolve
   - No `\textcolor{red}` TODO markers
2. **Second-pass anonymization.** Search the compiled PDF for:
   `grep -i "saml212\|samuellarson\|larson\|pebble" main.pdf`
   (needs `pdftotext` installed). Expected: zero matches.
3. **Reproducibility sanity.** Open `anonymous.4open.science/r/<your-slug>`
   in an incognito browser and verify:
   - `run_matrix_codi.py` is present
   - `README.md` describes what to run
   - No stray `.git` directories or author names in file paths
4. **Figures.** Open each PDF and confirm:
   - Captions are legible at target size (two-column)
   - Axis labels and tick labels are readable
   - No placeholder or "pending" annotations except where intentional
     (Fig.\ 3 scale sweep marks gpt2-large matrix as pending; Fig.\ 4 depth
     sweep marks n=16/32/64 as pending — both expected)
